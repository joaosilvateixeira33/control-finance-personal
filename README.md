## 💻 Sobre o Projeto

O **Nu Finance** (`control-finance-personal`) é uma aplicação web de gestão financeira desenvolvida para simplificar o acompanhamento diário de receitas e despesas. 

A aplicação foi construída com foco na experiência do usuário e em uma arquitetura limpa de componentes, utilizando recursos modernos do ecossistema React. Através do uso avançado de *React Hooks*, o projeto garante a reatividade instantânea na listagem de transações, no cálculo dinâmico de saldos e na persistência segura dos dados da aplicação.

---

## ✨ Funcionalidades

- **📊 Dashboard Dinâmico:** Visualização instantânea e cálculo automatizado do saldo atual, total de entradas e total de saídas.
- **➕ Gestão de Transações:** Adição simplificada de novas transações (receitas ou despesas) especificando descrição e valor.
- **📋 Listagem Detalhada:** Renderização de componentes para o acompanhamento do histórico de movimentações, com distinção visual clara entre créditos e débitos.
- **💾 Persistência de Dados:** Armazenamento local das informações cadastradas para garantir que o histórico financeiro seja mantido entre as sessões.
- **📱 Design Responsivo:** Interface totalmente adaptável e fluida para uso em smartphones, tablets e desktops.

## 🛠️ Tecnologias e Ferramentas

- **[React.js](https://react.dev/)** — Biblioteca base para a construção da interface e gerenciamento de estados com Hooks.
- **JavaScript (ES6+)** — Lógica de cálculo e gerenciamento de arrays/objetos.
- **Vite** — Build tool ultrarrápida para um ambiente de desenvolvimento ágil.
- **[pnpm](https://pnpm.io/)** — Gerenciador de dependências performático.
