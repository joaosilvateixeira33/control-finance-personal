## ⚙️ Como Executar o Projeto Localmente

Certifique-se de ter o Node.js instalado na sua máquina.

### 1. Clonar o Repositório
```bash
git clone https://github.com/joaosilvateixeira33/control-finance-personal.git
cd control-finance-personal
```

### 2. Instalar as Dependências
Utilizando o `pnpm` (recomendado):
```bash
pnpm install
```
*(Caso prefira, você também pode rodar `npm install` ou `yarn`).*

### 3. Iniciar o Servidor de Desenvolvimento
```bash
pnpm dev
```
O terminal irá gerar um link local (geralmente `http://localhost:5173`). Abra no seu navegador para começar a gerenciar suas finanças na aplicação!
